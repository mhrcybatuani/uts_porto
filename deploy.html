Tutorial: Cara Deploy Website ke GitHub Pages

Langkah-langkah Deploy:

  1. Siapkan Proyek Website: Buat folder dan tempatkan semua file HTML, CSS, dan gambar di dalamnya.
  2. Buat Repositori di GitHub: Login ke GitHub dan buat repositori baru.
  3. Upload File ke GitHub: Unggah file website ke repositori GitHub yang telah dibuat.
  4. Aktifkan GitHub Pages: Pada halaman repositori, buka tab `Settings` dan scroll ke bawah untuk mengaktifkan GitHub Pages pada branch `main`.
  5. Akses Website: GitHub Pages akan memberikan URL website kamu, misalnya `https://username.github.io/my-website/`.
